# 新闻热榜类

## 知乎热榜

### 功能介绍

查询实时知乎热榜并查看详情

### 触发指令

- ```知乎热榜```
- ```知乎热榜 + 编号```

> 查看相应编号文章详情，编号可通过知乎热榜获取
